# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-02-09

### Changed
- Made README more generic and domain-agnostic
- Removed trading-specific language from documentation
- Renamed concept references to "Category 1" and "Category 2" in docs
- Emphasized customizability for any domain (history, science, medicine, etc.)

### Added
- Explicit note that default config is for finance/ML but fully customizable
- Example showing how to customize for any domain

### Fixed
- PDF parsing now works correctly (pymupdf4llm properly detected)

## [1.0.0] - 2026-02-09

### Added
- Initial release
- Deep chapter-by-chapter analysis of epub and PDF files
- 1-10 rating system based on concept coverage
- Interactive mode with book selection
- Pre-configured for trading and machine learning concepts
- JSON and Markdown output formats
- Analysis logging to prevent duplicates
- Windows batch file for easy execution
